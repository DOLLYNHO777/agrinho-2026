# agrinho-2026

# 🌾 AgroForte • Futuro Sustentável

> **Projeto desenvolvido para o Programa Agrinho 2026** > Um ecossistema digital interativo projetado para conscientizar, educar e capacitar jovens e produtores rurais sobre as melhores práticas da agricultura sustentável no Brasil.

---

## 📌 Sobre o Projeto

O **AgroForte** é uma plataforma web responsiva que une tecnologia, gamificação e dados agronômicos reais. O objetivo principal é demonstrar que o equilíbrio entre a alta produtividade agrícola e a preservação ambiental não é apenas possível, mas essencial para o futuro do nosso planeta.

---

## 🚀 Funcionalidades Principais

* **🌱 Guia de Práticas Sustentáveis:** Apresentação didática de técnicas como Plantio Direto, Rotação de Culturas/Consórcio e Irrigação de Precisão.
* **📷 Galeria Real:** Demonstração visual do Agronegócio brasileiro sustentável (Soja, Café e Integração Lavoura-Pecuária-Floresta - ILPF).
* **🧪 Calculadora Avançada de Solo:** Uma ferramenta interativa baseada em parâmetros químicos reais (pH, Matéria Orgânica e Nitrogênio) que gera recomendações automáticas de manejo para culturas como soja, milho, café e cana-de-açúcar.
* **🎮 Minigame "Fazenda Sustentável":** Um simulador de estratégia em tempo real onde o usuário precisa gerenciar o plantio, irrigação, colheita e compostagem, equilibrando seus pontos com o índice de sustentabilidade da terra.
* **📊 Indicadores de Evolução:** Gráficos interativos que comparam o modelo de produção convencional com o sustentável, destacando a redução de emissões de $CO_2$ e o aumento de produtividade.
* **💬 Conexão com Especialistas:** Canal direto para simulação de suporte técnico com engenheiros agrônomos.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas de front-end para garantir performance, responsividade e uma interface visual atraente:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **Tailwind CSS:** Framework utilitário para um design moderno, fluido, responsivo e com suporte a efeitos de desfoque de fundo (*backdrop-blur*).
* **JavaScript (ES6+):** Lógica do minigame, processamento dos dados da calculadora de solo e manipulação dinâmica do DOM.
* **Chart.js (v4.4.1):** Biblioteca de alta performance utilizada para renderizar os gráficos de linha e barras de forma dinâmica.
* **Font Awesome (v6.6.0):** Conjunto de ícones vetoriais para enriquecimento visual da interface.
* **Google Fonts:** Integração das fontes *Inter* e *Playfair Display* para uma tipografia refinada.

---

## 💻 Como Executar o Projeto

Como o projeto foi desenvolvido de forma nativa (*Client-Side Only*), ele não necessita de instaladores ou servidores complexos:

1. Faça o clone deste repositório ou baixe o arquivo `_!DOCTYPE html_.html`.
2. Certifique-se de estar conectado à internet (necessário para carregar os estilos via CDN, gráficos e imagens).
3. Abra o arquivo `_!DOCTYPE html_.html` diretamente em qualquer navegador moderno (Chrome, Edge, Firefox ou Safari).

---

## 🎮 Regras do Minigame

Para obter o sucesso na sua **Fazenda Sustentável**, fique atento às mecânicas de tempo real:
* **Plantar (🌱):** Prepara a terra e inicia o cultivo.
* **Regar (💧):** Evolui a plantação para a fase de colheita e aumenta o seu índice de sustentabilidade.
* **Colher (✋):** Coleta os frutos maduros e gera uma grande quantidade de pontos.
* **Compostar (♻️):** Consome pontos para recuperar drasticamente a saúde e a sustentabilidade do solo.
* *Atenção:* A cada ciclo de tempo (6.5 segundos), a sustentabilidade da fazenda decresce naturalmente. Não deixe o solo esgotar!
